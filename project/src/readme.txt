Proyecto: Tienda en Línea
Instrucciones para el Asistente
Debes leer este archivo y realizar las modificaciones directamente en el código del proyecto según las tareas descritas a continuación.

Objetivo
Mejorar la interfaz y funcionalidad de la tienda en línea para hacerla más atractiva y funcional tanto para los usuarios como para los administradores.

Tareas a Realizar
1. Productos
Agrega más productos al catálogo existente.
Implementa la funcionalidad para mostrar reseñas de los productos.
Incluye un gráfico que muestre el historial de precios de cada producto.
2. Pantalla de Inicio
Implementa un carrusel que muestre los productos destacados.
Mejora el diseño de la pantalla de inicio haciéndola más atractiva visualmente:
Usa colores más llamativos y agradables.
Agrega más detalles visuales para captar la atención de los usuarios.
3. Interfaz de Usuario
Agrega un botón para que los usuarios puedan acceder a su perfil. Este botón debe estar ubicado al lado del botón de "Registrarse".
Crea una pantalla de perfil con opciones básicas de configuración.
4. Interfaz de Administrador
Crea una interfaz para administradores con las siguientes funcionalidades:
Agregar productos.
Eliminar productos.
Editar productos.
En la pantalla de inicio de sesión, agrega dos botones:
Uno para acceder a la versión de administrador.
Otro para acceder a la versión de usuario normal.
5. Diseño General
Haz la interfaz más atractiva para los usuarios, enfocándote en un diseño que invite a explorar los productos.
Agrega cualquier detalle adicional que consideres necesario para mejorar la experiencia del usuario, como nuevas pantallas o elementos visuales.
